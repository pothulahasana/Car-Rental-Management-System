# Car Rental Management System


The Car Rental Management System is a Java-based desktop application designed to automate and simplify car rental operations. It helps in managing customers, vehicles, bookings, and returns efficiently, reducing manual effort and errors.

The application is developed using Core Java (Swing) for the graphical user interface and JDBC for database connectivity.



##  Features- User Authentication (Admin Login)
- Car Management (Add, Update, Delete, View Cars)
- Customer Management
- Booking System
- Car Return Management
- Database Integration using JDBC
- View Rental Records

---

## Technologies Used
- Programming Language: Java
- GUI: Java Swing
- Database: Oracle / MySQL
- Connectivity: JDBC
